# Flora Haven - E-Commerce Website

Flora Haven is a modern and responsive e-commerce website designed to provide users with a seamless online shopping experience for plants, decorative vases, and nature-inspired perfumes. The platform combines aesthetics and functionality to create an engaging shopping environment while promoting eco-friendly living and a connection with nature.

## Features

* Responsive and user-friendly design
* Product categories for Plants, Vases, and Perfumes
* Modern homepage with interactive sections
* Customer account management page
* Order tracking functionality
* Product recommendations
* Profile editing features
* Smooth navigation and responsive layouts

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Highlights

* Nature-inspired user interface
* Clean and modern design
* Mobile-friendly layout
* Interactive shopping experience
* Eco-conscious branding and theme

## Future Enhancements

* Shopping cart functionality
* Secure user authentication
* Product search and filtering

## About Flora Haven

Flora Haven is inspired by the idea of bringing nature, beauty, and freshness into every home through carefully selected plants, artisan vases, and botanical fragrances.
